# Car Rental Management System 🚗

## 📖 Project Overview
This project is a comprehensive **Car Rental Management System** developed as part of a dedicated practice and training journey. The primary goal of this project is to apply advanced programming concepts and transform logical business requirements into a functional, scalable technical system.

---

## 🎯 Project Goal
This is a **Practice Project** designed for educational and skill-validation purposes. It focuses on:
* **Mastering the .NET environment** using the C# language.
* **Implementing Object-Oriented Programming (OOP)** principles.
* **Practicing Separation of Concerns** through tiered architecture.

---

## 🛠 Development Lifecycle & My Role
I followed a clear engineering methodology to build this system, which involved:

1.  **System Analysis:** Analyzing the core business logic, identifying essential entities (Customers, Vehicles, Bookings), and defining the necessary business rules for a rental environment.
2.  **Database Design & Implementation:** Designing the relational schema and building the database on **SQL Server** to ensure data integrity and efficient relationships.
3.  **System Implementation:** Developing the system layers from the ground up, ensuring seamless communication between the UI and the data, while focusing on writing **Clean Code**.

---

## 💻 Technical Stack
The system was built using the following technologies and standards:

* **Architecture:** **3-Tier Architecture** (Presentation, Business Logic, and Data Access Layers) to ensure a modular and maintainable codebase.
* **Programming Language:** C# (.NET Framework).
* **Database:** Microsoft SQL Server.
* **Version Control:** Git & GitHub.

---

## ✨ Key Features
* **Comprehensive Management:** Full control over Customers and Vehicles data.
* **Operation Efficiency:** Smooth handling of Rental and Return operations.
* **Automated Logic:** Automated calculation of costs and rental durations.
* **Monitoring:** Integrated **Event Log** monitoring for system tracking.
